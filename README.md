# 🏋️ GymTracker MVP

GymTracker es una aplicación web para registrar entrenamientos de gimnasio de forma simple y rápida. Permite crear sesiones de entrenamiento, añadir ejercicios con series, repeticiones y peso, y consultar el historial completo.

El objetivo del proyecto es practicar desarrollo frontend moderno con React + TypeScript, gestión de estado sin librerías externas y persistencia local con localStorage.
